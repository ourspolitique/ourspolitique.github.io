## 一、政治学

中国政治学者潘维认为，政治学在第二次世界大战后不断演化，形成六大专业方向： （1）本国政治； （2）比较政治（亦称“世界政治”，意即“外国政治”）； （3）国际政治（亦称“国际关系”）； （4）政治哲学（亦称“政治理论”）； （5）政治学研究方法； （6）行政管理。
